蓝桥杯14-4:排列序数

发表于 2017-05-23

标题：排列序数

如果用a b c d这4个字母组成一个串，有4!=24种，如果把它们排个序，每个串都对应一个序号：
abcd 0
abdc 1
acbd 2
acdb 3
adbc 4
adcb 5
bacd 6
badc 7
bcad 8
bcda 9
bdac 10
bdca 11
cabd 12
cadb 13
cbad 14
cbda 15
cdab 16
cdba 17
…

现在有不多于10个两两不同的小写字母，给出它们组成的串，你能求出该串在所有排列中的序号吗？

【输入格式】
一行，一个串。

【输出格式】
一行，一个整数，表示该串在其字母所有排列生成的串中的序号。注意：最小的序号是0。

例如：
输入：
bdca

程序应该输出：
11

再例如：
输入：
cedab

程序应该输出：
70

资源约定：
峰值内存消耗（含虚拟机） < 256M
CPU消耗 < 1000ms

请严格按要求输出，不要画蛇添足地打印类似：“请您输入…” 的多余内容。

所有代码放在同一个源文件中，调试通过后，拷贝提交该源码。
注意：不要使用package语句。不要使用jdk1.7及以上版本的特性。
注意：主类的名字必须是：Main，否则按无效代码处理。

import java.util.Scanner;

public class c14 {

/**
 * @param args
 */
private static char b[];//存放输入字符
private static char c[] ={'a','b','c','d','e','f','g','h','i','j'};//遍历字符数组
private static boolean s[] = new boolean [10];//布尔数值
private static int count = 0;
public static void main(String[] args) {
    // TODO Auto-generated method stub
    Scanner in = new Scanner(System.in);
    String str = in.next();
    char[] a = new char[str.length()];//遍历形成的数组
    b = new char[str.length()];
    for(int i=0;i<str.length();i++)
        b[i]=str.charAt(i);
    dfs(a,0,str.length());
}
public static void dfs(char[] a,int n,int s1){
    if(n>=s1){
        int m=0;
        for(int i=0;i<s1;i++)
            if(a[i]!=b[i]){//遍历形成的数组与输入数组比较，相等则输出，不想等，count+1继续遍历
                m=1;
                break;
            }
        if(m==1){
            count++;
            }
        else if(m==0)
            System.out.print(count);
    }
    else{
        for(int i=0;i<s1;i++){
            if(s[i]==false){//回溯
                s[i]=true;
                a[n]=c[i];
                dfs(a,n+1,s1);
                s[i]=false;
            }
        }
    }
}

}

dfs算法一定要会，运用回溯的思想，层层遍历判断
