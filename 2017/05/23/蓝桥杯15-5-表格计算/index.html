蓝桥杯15-5:表格计算

发表于 2017-05-23 
标题：表格计算

某次无聊中， atm 发现了一个很老的程序。这个程序的功能类似于 Excel ，它对一个表格进行操作。
不妨设表格有 n 行，每行有 m 个格子。
每个格子的内容可以是一个正整数，也可以是一个公式。
公式包括三种：
1. SUM(x1,y1:x2,y2) 表示求左上角是第 x1 行第 y1 个格子，右下角是第 x2 行第 y2 个格子这个矩形内所有格子的值的和。
2. AVG(x1,y1:x2,y2) 表示求左上角是第 x1 行第 y1 个格子，右下角是第 x2 行第 y2 个格子这个矩形内所有格子的值的平均数。
3. STD(x1,y1:x2,y2) 表示求左上角是第 x1 行第 y1 个格子，右下角是第 x2 行第 y2 个格子这个矩形内所有格子的值的标准差。

标准差即为方差的平方根。
方差就是：每个数据与平均值的差的平方的平均值，用来衡量单个数据离开平均数的程度。

公式都不会出现嵌套。

如果这个格子内是一个数，则这个格子的值等于这个数，否则这个格子的值等于格子公式求值结果。

输入这个表格后，程序会输出每个格子的值。atm 觉得这个程序很好玩，他也想实现一下这个程序。

「输入格式」
第一行两个数 n, m 。
接下来 n 行输入一个表格。每行 m 个由空格隔开的字符串，分别表示对应格子的内容。
输入保证不会出现循环依赖的情况，即不会出现两个格子 a 和 b 使得 a 的值依赖 b 的值且 b 的值依赖 a 的值。

「输出格式」
输出一个表格，共 n 行，每行 m 个保留两位小数的实数。
数据保证不会有格子的值超过 1e6 。

「样例输入」
3 2
1 SUM(2,1:3,1)
2 AVG(1,1:1,2)
SUM(1,1:2,1) STD(1,1:2,2)

「样例输出」
1.00 5.00
2.00 3.00
3.00 1.48

「数据范围」
对于 30% 的数据，满足： n, m <= 5
对于 100% 的数据，满足： n, m <= 50


资源约定：
峰值内存消耗（含虚拟机） < 512M
CPU消耗  < 2000ms


请严格按要求输出，不要画蛇添足地打印类似：“请您输入...” 的多余内容。

所有代码放在同一个源文件中，调试通过后，拷贝提交该源码。
注意：不要使用package语句。不要使用jdk1.7及以上版本的特性。
注意：主类的名字必须是：Main，否则按无效代码处理。

import java.util.Scanner;


public class b5 {

    /**
     * @param args
     */
    private static String[][] b ;
    private static String[][] str;
    public static void main(String[] args) {
        // TODO Auto-generated method stub
        Scanner in = new Scanner (System.in);
        int n = in.nextInt();
        int m = in.nextInt();
        str = new String[n][m];
        b = new String[n][m];
        for(int i=0;i<n;i++)
            for(int j=0;j<m;j++)
                str[i][j]=in.next();
        f(n,m);

    }
    public static void f(int n,int m){
        java.text.DecimalFormat   df   =new   java.text.DecimalFormat("#.00");  //保留两位小数
        for(int i=0;i<n;i++)
            for(int j=0;j<m;j++){
                if(str[i][j].length()<=2)b[i][j]=df.format((double)Integer.parseInt(str[i][j]));
                else if(str[i][j].substring(0, 3).equals("SUM"))b[i][j]=df.format((double)SUM(i,j));
                else if(str[i][j].substring(0, 3).equals("AVG"))b[i][j]=df.format((double)AVG(i,j));
                else if(str[i][j].substring(0, 3).equals("STD"))b[i][j]=df.format((double)STD(i,j));
            }
        for(int i=0;i<n;i++){
            for(int j=0;j<m;j++)
                System.out.print(b[i][j]+" ");
            System.out.println();
        }
    }
    public static double SUM(int x,int y){
        String str1 = "",str2="";
        for(int i=4;i<str[x][y].length();i++){
            if(str[x][y].substring(i, i+1).equals(":")){ //根据:分离
                str1=str[x][y].substring(4, i);
                str2=str[x][y].substring(i+1, str[x][y].length());
                break;
            }    
        }
        int a1 = 0,b1 = 0,a2 = 0,b2 = 0,j=0;
        while(!str1.substring(j,j+1).equals(",")){//根据，分离
            j++;
        }
        a1=Integer.parseInt(str1.substring(0, j))-1;
        b1=Integer.parseInt(str1.substring(j+1, str1.length()))-1;
        j=0;
        while(!str2.substring(j,j+1).equals(",")){
            j++;
        }
        a2=Integer.parseInt(str2.substring(0, j))-1;
        b2=Integer.parseInt(str2.substring(j+1, str2.length()-1))-1; //str2 里面的字符串多带一个）
        double sun = 0;
        for(int n=a1;n<=a2;n++)
            for(int m=b1;m<=b2;m++){
                if(str[n][m].length()<=2)sun+=Integer.parseInt(str[n][m]);
                else if(str[n][m].substring(0, 3).equals("SUM"))sun+=SUM(n,m);
                else if(str[n][m].substring(0, 3).equals("AVG"))sun+=AVG(n,m);
                else if(str[n][m].substring(0, 3).equals("STD"))sun+=STD(n,m);
            }
        return sun;
    }
    public static double  AVG(int x,int y){
        String str1 = "",str2="";
        for(int i=4;i<str[x][y].length();i++){
            if(str[x][y].substring(i, i+1).equals(":")){
                str1=str[x][y].substring(4, i);
                str2=str[x][y].substring(i+1, str[x][y].length());
                break;
            }    
        }
        int a1 = 0,b1 = 0,a2 = 0,b2 = 0,j=0;
        while(!str1.substring(j,j+1).equals(",")){
            j++;
        }
        a1=Integer.parseInt(str1.substring(0, j))-1;
        b1=Integer.parseInt(str1.substring(j+1, str1.length()))-1;
        j=0;
        while(!str2.substring(j,j+1).equals(",")){
            j++;
        }
        a2=Integer.parseInt(str2.substring(0, j))-1;
        b2=Integer.parseInt(str2.substring(j+1, str2.length()-1))-1;
        double avg = 0,sun=0 ,count=(a2-a1+1)*(b2-b1+1);
        for(int n=a1;n<=a2;n++)
            for(int m=b1;m<=b2;m++){
                if(str[n][m].length()<=2)sun+=Integer.parseInt(str[n][m]);
                else if(str[n][m].substring(0, 3).equals("SUM"))sun+=SUM(n,m);
                else if(str[n][m].substring(0, 3).equals("AVG"))sun+=AVG(n,m);
                else if(str[n][m].substring(0, 3).equals("STD"))sun+=STD(n,m);
            }
        avg = sun/count;
        return avg;
    }
    public static double  STD(int x,int y){
        String str1 = "",str2="";
        for(int i=4;i<str[x][y].length();i++){
            if(str[x][y].substring(i, i+1).equals(":")){
                str1=str[x][y].substring(4, i);
                str2=str[x][y].substring(i+1, str[x][y].length());
                break;
            }    
        }
        int a1 = 0,b1 = 0,a2 = 0,b2 = 0,j=0;
        while(!str1.substring(j,j+1).equals(",")){
            j++;
        }
        a1=Integer.parseInt(str1.substring(0, j))-1;
        b1=Integer.parseInt(str1.substring(j+1, str1.length()))-1;
        j=0;
        while(!str2.substring(j,j+1).equals(",")){
            j++;
        }
        a2=Integer.parseInt(str2.substring(0, j))-1;
        b2=Integer.parseInt(str2.substring(j+1, str2.length()-1))-1;
        double avg=AVG(x,y),sun=0,count=(a2-a1+1)*(b2-b1+1),std=0;;
        for(int n=a1;n<=a2;n++)
            for(int m=b1;m<=b2;m++){
                if(str[n][m].length()<=2)sun+=(Integer.parseInt(str[n][m])-avg)*(Integer.parseInt(str[n][m])-avg);
                else if(str[n][m].substring(0, 3).equals("SUM"))sun+=(SUM(n,m)-avg)*(SUM(n,m)-avg);
                else if(str[n][m].substring(0, 3).equals("AVG"))sun+=(AVG(n,m)-avg)*(AVG(n,m)-avg);
                else if(str[n][m].substring(0, 3).equals("STD"))sun+=(STD(n,m)-avg)*(STD(n,m)-avg);
            }
        std = Math.sqrt(sun/count);
        return std;
    }

}

没有什么算法难度，主要注意分离字符判断递归即可
